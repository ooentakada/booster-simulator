# 応援共創シミュレーター 共有用

30人を仲間と集客するためのミニシミュレーションです。

## 公開方法

一番簡単なのは Netlify Drop です。

1. `public` フォルダを開く
2. https://app.netlify.com/drop にアクセス
3. `public` フォルダをドラッグ&ドロップ
4. 発行されたURLをメンバーに共有

GitHub Pages、Vercel、Cloudflare Pagesでも公開できます。公開対象は `public` フォルダの中身だけです。

## フィードバック依頼文

以下をメンバーに送ると使いやすいです。

```text
BOOSTERの応援共創シミュレーターのプロトタイプを作っています。
5分くらいで遊べるので、率直なフィードバックをください。

見てほしい観点:
1. 面白かったところ
2. わかりにくかったところ
3. BOOSTERらしい/らしくないと感じたところ
4. もう一度やりたいと思うか
5. 実際のイベント集客や仲間づくりに近いと感じたか

最後の結果画面で「結果をコピー」を押すと、結果とフィードバック観点を貼れます。
```

## ファイル

- `styles.css`: 見た目
